# Site Delícias Gourmet

Este repositório reúne os materiais do projeto acadêmico Site Delícias Gourmet, desenvolvido para as disciplinas de Engenharia de Requisitos, Design e Interface Gráfica e Modelagem de Sites Básicos.

O objetivo do projeto é criar uma landing page institucional para a empresa Delícias Gourmet, centralizando informações importantes para o cliente, como história da empresa, cardápio, localização, horário de funcionamento e formas de contato.

O projeto também contempla o planejamento no Figma, a definição de requisitos funcionais e não funcionais, e a futura implementação do site utilizando HTML5, CSS3.
